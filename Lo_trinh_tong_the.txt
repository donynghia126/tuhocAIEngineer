LỘ TRÌNH AI ENGINEER 2025 (PHIÊN BẢN CHUYÊN SÂU & MỞ RỘNG)

MỤC TIÊU TỔNG QUAN:

Trở thành AI Engineer có tư duy sản phẩm (Product Mindset).

Nắm vững Generative AI, AI Agents và quy trình MLOps thực chiến.

Thời gian học: 1.5 – 2 tiếng/ngày (Duy trì bền bỉ).

Tổng thời gian ước tính: ~14-16 tháng (Học sâu hiểu kỹ).

GIAI ĐOẠN 1: NỀN TẢNG - TOÁN, PYTHON & COMPUTER SCIENCE

Thời gian: 8 - 9 Tuần

Mục tiêu: Xây dựng nền móng vững chắc, không học vẹt code.

Chủ đề cần học:

Toán học (Học theo kiểu Just-in-time: cần dùng mới học sâu):

Đại số tuyến tính (Linear Algebra): Ma trận, Vector.

Giải tích (Calculus): Đạo hàm, Gradient Descent.

Xác suất thống kê cơ bản.

Lập trình Python nâng cao:

OOP, Decorators, Generators.

Cách viết code sạch (Clean Code).

Cấu trúc dữ liệu & Giải thuật (DSA):

Array, Hashmap, Graph cơ bản.

Git & GitHub flow: Quản lý source code.

Dự án thực hành:

[Project 1] Mini Numpy from Scratch:

Tự viết tay 5 hàm: dot product, mean, std, normalize, softmax.

Mục đích: Hiểu bản chất tính toán bên dưới thư viện.

[Project 2] CLI Student Manager:

Quản lý học sinh (CRUD) lưu file JSON/SQLite.

GIAI ĐOẠN 2: DATA SCIENCE & FEATURE ENGINEERING

Thời gian: 7 - 8 Tuần

Mục tiêu: Thành thạo kỹ năng xử lý dữ liệu (chiếm 80% công việc thực tế).

Chủ đề cần học:

Thư viện xử lý dữ liệu: Pandas & Polars (Polars đang là xu hướng).

Thư viện tính toán & vẽ biểu đồ: NumPy, Matplotlib, Seaborn.

SQL nâng cao: Window Functions, CTEs.

Feature Engineering: Xử lý dữ liệu thiếu (Imputation), Mã hóa (Encoding), Chuẩn hóa (Scaling).

Dự án thực hành:

[Project] Phân tích dữ liệu Thương mại điện tử (EDA):

Dữ liệu: Shopee/Tiki hoặc Retail dataset.

Nhiệm vụ: Làm sạch data -> Trực quan hóa -> Tìm Insight -> Viết báo cáo kể chuyện (Storytelling).

GIAI ĐOẠN 3: MACHINE LEARNING CỔ ĐIỂN (THỰC CHIẾN)

Thời gian: 10 - 12 Tuần

Mục tiêu: Nắm vững các thuật toán cốt lõi cho dữ liệu bảng (Tabular Data).

Chủ đề cần học:

Supervised Learning: Linear/Logistic Regression, Decision Trees.

Ensemble Learning (Cực quan trọng): Random Forest, XGBoost, LightGBM.

Unsupervised Learning: K-Means, DBSCAN, PCA.

Đánh giá Model: Confusion Matrix, ROC-AUC, F1-Score.

Explainable AI: Hiểu model hoạt động thế nào bằng SHAP values.

Dự án thực hành:

[Project] Credit Scoring / Fraud Detection (Chấm điểm tín dụng/Phát hiện gian lận):

Nhiệm vụ: Xử lý mất cân bằng dữ liệu (SMOTE), Train model XGBoost/LightGBM, Giải thích kết quả bằng SHAP.

GIAI ĐOẠN 4: DEEP LEARNING FOUNDATION

Thời gian: 14 - 16 Tuần

Mục tiêu: Bước chân vào thế giới Neural Networks và xử lý ảnh/ngôn ngữ.

Chủ đề cần học:

Neural Networks & Backpropagation (Lan truyền ngược).

Framework: PyTorch (Phải nắm thật vững).

Computer Vision (CNN): ResNet, YOLO (Object Detection).

Sequence Data (RNN): LSTM/GRU.

Cơ chế Attention & Transformers: Cốt lõi để hiểu LLM sau này.

Dự án thực hành:

[Project 1] Custom Object Detection:

Fine-tune YOLOv8/v10 để nhận diện biển số xe hoặc mũ bảo hiểm.

[Project 2] Sentiment Analysis:

So sánh hiệu năng giữa LSTM và BERT model.

GIAI ĐOẠN 5: LLM, RAG & AI AGENTS (TRỌNG TÂM CÔNG NGHỆ MỚI)

Thời gian: 18 - 20 Tuần

Mục tiêu: Làm chủ công nghệ Generative AI đang được săn đón nhất.

Chủ đề cần học:

Advanced RAG:

Hybrid Search (Kết hợp từ khóa + Vector).

Re-ranking (Sắp xếp lại kết quả tìm kiếm để tăng độ chính xác).

Vector Databases: ChromaDB, Qdrant hoặc Weaviate.

Frameworks: LangChain & LlamaIndex.

AI Agents (Tương lai của AI):

LangGraph (Quản lý luồng tác vụ phức tạp).

Tool calling, ReAct pattern.

LLM Evaluation: Kiểm thử Chatbot bằng Ragas, DeepEval.

Dự án thực hành:

[Project 1] Advanced RAG Chatbot for Legal/Docs:

Xây dựng chatbot tra cứu luật/tài liệu nội bộ với độ chính xác cao nhờ Re-ranking.

[Project 2] AI Agent Personal Assistant:

Xây dựng Agent biết tự Google Search -> Tóm tắt -> Viết email nháp -> Báo cáo lại cho người dùng.

GIAI ĐOẠN 6: SOFTWARE ENGINEERING & DATA OPS

Thời gian: 10 - 12 Tuần

Mục tiêu: Không chỉ biết train model, phải biết đóng gói thành sản phẩm.

Chủ đề cần học:

Backend for AI: FastAPI (Viết API bất đồng bộ, Websockets).

Containerization: Docker (Viết Dockerfile tối ưu cho GPU).

Orchestration: Kubernetes cơ bản (Pod, Service, Deployment).

CI/CD: GitHub Actions (Tự động test và build code).

Database: PostgreSQL (User data), Redis (Cache lịch sử chat).

Dự án thực hành:

[Project] Scalable AI API Service:

Xây dựng API chat có tính năng Streaming response (chữ chạy từng dòng).

Đóng gói vào Docker và thiết lập quy trình CI/CD cơ bản.

GIAI ĐOẠN 7: MLOPS & TỐI ƯU HÓA HỆ THỐNG

Thời gian: 10 - 12 Tuần

Mục tiêu: Triển khai model ra môi trường thực tế (Production) với chi phí tối ưu.

Chủ đề cần học:

Model Serving: vLLM, TGI (Tối ưu tốc độ sinh text).

Quantization (Lượng tử hóa): GGUF, GPTQ, AWQ (Chạy model lớn trên phần cứng yếu/chi phí thấp).

Experiment Tracking: MLflow hoặc Weights & Biases.

Cloud (Tập trung AWS): SageMaker, Lambda, EC2, S3.

Monitoring: Prometheus + Grafana (Theo dõi độ trễ, tài nguyên).

Dự án thực hành:

[Capstone Project] End-to-End LLM System on Cloud:

Tự Quantize model Llama-3 xuống 4-bit.

Deploy lên AWS EC2.

Xây dựng giao diện web (Streamlit).

Gắn Dashboard theo dõi sức khỏe hệ thống.

GIAI ĐOẠN 8: CHUẨN BỊ SỰ NGHIỆP (SONG SONG GIAI ĐOẠN CUỐI)

Thời gian: Ongoing

Nhiệm vụ:

Luyện thuật toán: LeetCode (Mức độ Medium - Array/String/DP).

Phỏng vấn thử: Luyện System Design cho các hệ thống AI/ML.

(Tùy chọn) Chứng chỉ: AWS Certified Machine Learning Specialty.
